## App Description
This app will help users better understand their academic preformance by combining GPA calculations with planning and analysis tools. Users will be able to explore different grade outcomes, estimate what grades they need to reach a target GPA, and identify which classes are helping or hurting their GPA the most. The app is designed to make GPA tracking more interactive and useful than a basic calculator.

## Instructions 
- Input user classes(quantity and class name), grade, credit hours, and planned credits in the future to calculate current GPA and AVG GPA needed to maintain "XXX" GPA. 

## Intended Users
This app is intended for students (high school and college) who want to uderstand how individual classes affect their grades, waht classes tehy need to improve on in order to reach a target GPA, and plan for overall improved academic preformance.

- Grade Impact Analyzer
